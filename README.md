# Trees

##  **Описание проекта**

В этом проекте реализовано простое дерево решений (**CART**) с использованием **индекса Джини**.

Алгоритм автоматически:
- выбирает лучший признак  
- находит оптимальный порог разделения  
- строит дерево решений  

Код написан “с нуля”, без использования готовых библиотек машинного обучения.

---

##  **Как работает алгоритм**

### **1. Индекс Джини**

Используется для оценки “чистоты” данных.

- Чем ближе к **0** → тем лучше  
- Чем ближе к **0.5** → тем хуже (данные перемешаны)

Формула:
Gini = 1 - (p1² + p0²)

---

### **2. Разделение данных**

Данные делятся по условию:
feature < threshold

- Левая часть → значения меньше порога  
- Правая часть → значения больше или равны  

---

### **3. Поиск лучшего разбиения**

Алгоритм:
- перебирает все признаки  
- берёт уникальные значения  
- строит возможные пороги (середины)  
- считает Gini после разделения  
- выбирает вариант с максимальным **Gain**

---

### **4. Построение дерева**

Дерево строится рекурсивно:

- если данные “чистые” → создаётся лист  
- если достигнута максимальная глубина → остановка  
- иначе → ищется лучшее разделение и процесс повторяется  

---

## **Что делает код**

Основные функции:

### **calculate_gini**
Считает индекс Джини для набора значений.

### **split_dataset**
Делит DataFrame на две части по условию.

### **find_best_split**
Ищет:
- лучший признак  
- лучший порог  
- максимальное улучшение (gain)

### **build_tree**
Рекурсивно строит дерево решений.

### **print_tree**
Выводит дерево в читаемом виде.
